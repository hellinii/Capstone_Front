### 1.4 제약사항

본 시험결과서에서 기술한 내용은 다음의 전제 및 명시적 한계를 지닌다.

1. **데이터 기반 평가의 국한성:** 본 시험은 평가 대상 모델 자체가 아닌, 의뢰자가 당사 평가 시스템에 업로드한 추론 결과 데이터셋(Inference Result Dataset)을 기반으로 정량적 지표를 산출하였다. 따라서 본 결과서는 모델의 내부 아키텍처, 소스코드, 원본 학습 데이터의 무결성에 대한 직접적인 검증 결과를 포함하지 않는다.
2. **결과서 효력의 한계:** 산출된 평가지표 및 종합 소견은 의뢰자가 제출한 특정 테스트 데이터셋 환경에 국한된 성능을 증명하며, 실제 서비스 배포 이후 발생하는 데이터 분포 변화(Data Drift)나 환경적 요인에 따른 성능 변동을 보장하지 않는다.
3. **자동화 진단 소견에 대한 면책:** 종합 분석 소견은 산출된 통계 지표를 바탕으로 한 성능 해석 및 기술적 권고안으로, 모델 개선을 위한 보조적 참고 자료이다. 최종적인 서비스 운영 및 상용화 의사결정의 책임은 전적으로 평가 의뢰자에게 있다.
4. **LLM 생성 소견에 대한 면책:** 본 결과서의 서술형 분석 소견(7절 이후)은 산출된 정량 지표를 바탕으로 LLM(Large Language Model) 및 RAG(Retrieval-Augmented Generation) 기술을 활용하여 자동 생성되었다. 해당 소견은 참고용이며, 전문가의 추가 검토를 권장한다.

## 5. 시험항목 및 방법

사용자가 선택한 시험항목에 대해서만 평가를 수행한다.

### 5.1 공통 지표 *(Binary / Multi-class / Multi-label 모두 적용)*

| Metric ID | 시험항목 | 시험목표 및 기준 | 시험방법 및 산정식 |
| --- | --- | --- | --- |
| M1 | Accuracy | 전체 데이터셋에 대해 예측값과 실제 라벨의 일치율을 측정하여 전반적인 분류 정확도를 평가함 | (TP + TN) / (TP + FP + TN + FN) × 100 [%] |
| M2 | Precision | 양성 예측 중 실제 양성의 비율을 측정하여 오탐지(FP) 비용 및 예측 정밀성을 검증함 | TP / (TP + FP) |
| M3 | Recall | 실제 양성 중 탐지 비율을 산출하여 미탐지(FN) 방지 능력을 확인함 | TP / (TP + FN) |
| M4 | F1 Score | 정밀도와 재현율의 조화 평균으로 균형 잡힌 성능을 대표함 | 2TP / (2TP + FP + FN) |
| M21 | Confusion Matrix | 실제/예측 클래스 간 교차 분포를 행렬화하여 오분류 경향을 진단함 | 실제값 대비 예측값 매칭 카운트 산출 + 히트맵 자동 삽입 |
| M22 | Class별 Metric | 각 클래스별 독립적인 P/R/F1 지표를 산출하여 취약 클래스를 분석함 | 개별 클래스별 P/R/F1 산출 |
| M23 | Imbalance Ratio | 클래스별 샘플 편차를 수치화하여 데이터 불균형의 영향을 사전 확인함 | max(class count) / min(class count) |
